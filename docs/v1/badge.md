#### badge徽章
徽章组件一般放在`tab-item`里面，或者`title`，提示有多少条未读消息！`badge`只有一种颜色，就是红色。
<div class="code-pre">
  <span class="hc-badge">2</span>
  <span class="hc-badge">15</span>
  <span class="hc-badge">99+</span>
  <span class="hc-badge">999+</span>
</div>
